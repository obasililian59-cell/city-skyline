# city-skyline

A responsive city skyline built with pure HTML and CSS.  
This project explores layout structure, depth layering, and intentional color systems using CSS variables.

✨ Features :
- Fully responsive layout (mobile → desktop tested)
- Multiple architectural styles:
  - Flat rooftops  
  - Stepped "wedding cake" roof  
  - Triangular roof using CSS borders  
  - Rounded architectural caps  
- Satellite dish detail using CSS transforms
- Layered depth with intentional z-index management
- CSS variable-based color system for scalable palette control
- Media query adjustments for larger screens

🎨 Design Approach :
The project was built using a structure-first methodology:
1. Define layout structure
2. Add architectural details
3. Apply consistent color systems via CSS variables
4. Refine spacing and depth
5. Adjust for responsive breakpoints

🛠️ Built With :
- HTML5  
- CSS3 (Flexbox, Gradients, Media Queries, CSS Variables, Transforms)
